为什么拦截器特别适合做登录校验

拦截器适合放在这里，不只是因为“会拦请求”，更因为它离 Spring MVC 的控制层语义更近。很多登录放行、页面跳转、资源排除规则，放在拦截器里都比放在更底层的过滤器里自然。
