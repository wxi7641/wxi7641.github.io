Token 和 JWT 到底是什么关系

Token 是广义概念，JWT 是一种具体实现形式。

概念 含义
Token 令牌，代表某种认证凭证
JWT 一种结构化 Token，包含头、载荷、签名

JWT 最大的特点是：认证信息被编码进 token 本身，再通过签名防止被篡改。
