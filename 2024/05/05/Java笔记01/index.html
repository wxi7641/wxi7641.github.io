一.java简介

1. Java发展史

  • 1990 sun公司 启动 绿色计划
  • 1992 创建 oak(橡树)语言 -> java
  • 1994 gosling 参加硅谷大会 演示java功能,震惊世界
  • 1995 sun 正式发布java第1个版本
  • 2009年，甲骨文公司宣布收购Sun
  • 2014，发布java8

jdk官网：Java Downloads | Oracle

2. Java技术体系

随着Java的发展，SUN给Java又分出了三个不同版本：

  • Java SE：Standard Edition（标准版）核心

    • 支持面向桌面的应用，JVM虚拟机/标准库

  • Java EE：Enterprise Edition（企业版）

    • Web页面的开发，在SE的基础上加了大量的API和库；

  • Java ME：Micro Edition（小型版）

    • 移动端的开发

    联系：

3.Java特性

Java介于编译型语言和解释型语言之间（字节码文件）

  1. 简单易学；
  2. 面向对象（封装，继承，多态）（OOP原则）；
  3. 跨平台性（ Java 虚拟机实现跨平台性）；
  4. 支持多线程（ C++ 语言没有内置的多线程机制，因此必须调用操作系统的多线程功能来进行多线程程序设计，而 Java 语言却提供了多线程支持）；
  5. 稳健性（不支持指针、垃圾回收）；
  6. 支持网络编程并且很方便（ Java 语言诞生本身就是为简化网络编程设计的，因此 Java 语言不仅支持网络编程而且很方便）；
  7. 编译与解释并存（C、C++；Python、PHP）；

（解释器就是jvm虚拟机）

4.JVM vs JDK vs JRE

（1）JVM

JAVA的一个虚拟机（JVM），是用来运行JAVA字节码文件的

  • JVM是实现跨平台的本质，“一次编写，到处使用”
  • 支持 Windows 、 macOS 、Linux

（2）JRE

JRE 是 JAVA 文件的运行环境

1. 不能够去创建新的程序
1. JVM、java类库、java命令等
1. 如果只运行JAVA程序，有JRE就够了

（3）JDK.

JDK 是功能齐全的 Java SDK ，并且拥有 JRE 的一切功能

  • 编译器（javac）
  • 工具（javadoc）
    • 他能够创建新的程序
